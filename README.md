# Vue 3 + Typescript + Vite + Pinia + SSR

[Vue SSR 指南](https://ssr.vuejs.org/zh/)

[Vite SSR](https://cn.vitejs.dev/guide/ssr.html)

[官方 demo](https://github.com/vitejs/vite/tree/main/packages/playground/ssr-vue)



## 开发

```
// npm
npm run dev
// yarn
yarn dev
```

## 打包

```
// npm
npm run build
// yarn
yarn build
```

## 启动服务

```
// npm
npm run server
// yarn
yarn server
```
